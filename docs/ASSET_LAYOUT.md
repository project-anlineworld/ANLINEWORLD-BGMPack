# フォルダ方針

- `assets/minecraft/sounds/anlineworld/bgm/` : 長時間再生するBGM
- `assets/minecraft/sounds/anlineworld/sfx/` : 短い効果音
- すべての再生用ファイル名は **英小文字・数字・アンダースコア** に統一
- 音源はMinecraft互換の **OGG/Vorbis** に統一
- `assets/mmoinventory/` と `assets/space/` は導入済みプラグインとの参照を壊さないため配置を変更しない

このパックのZIP最上位には `pack.mcmeta` と `assets/` が直接存在します。さらに親フォルダで包み直さないでください。
